Shopping Cart

I developed a Java-based GUI shopping cart application that simulates an online store experience. The program reads a list of items from a file, allows users to add items to their cart by entering an item ID, and generates a receipt upon checkout. To ensure accountability, every transaction is recorded in a transaction file for documentation and review.

Features

  • GUI Interface: Provides an intuitive and user-friendly shopping experience.
  • Item Lookup: Reads product data from a file and matches items by their unique IDs.
  • Shopping Cart Management: Users can add items to their cart and view their selections in real time.
  • Checkout & Receipt: Automatically generates and displays a receipt with item details and totals.
  • Transaction Logging: All purchases are written to a transaction file for record-keeping.
  • Bulk Discounts: Automatically applies discounts when items are purchased in bulk.
>

Challenges

  • File Handling: Ensuring that inventory and transaction files were read and written correctly without data corruption.
  • Error Handling: Managing invalid inputs, such as incorrect item IDs or empty carts.
  • GUI Responsiveness: Designing a layout that remained clear and functional as the cart updated in real time.
